# Galericim - Windows Uygulaması

## Kurulum ve Çalıştırma

Bu paket Galericim uygulamasının Windows için hazırlanmış çalıştırılabilir versiyonudur.

### Sistem Gereksinimleri:
- Windows 10 veya üzeri
- Microsoft Visual C++ Redistributable (genellikle zaten yüklüdür)

### Çalıştırma:
1. 'galericim.exe' dosyasına çift tıklayın
2. Uygulama otomatik olarak başlayacaktır

### Dosya Yapısı:
- galericim.exe: Ana uygulama dosyası
- flutter_windows.dll: Flutter runtime kütüphanesi
- data/: Uygulama veri dosyaları

### Sorun Giderme:
- Eğer uygulama açılmıyorsa, Microsoft Visual C++ Redistributable'ın en son versiyonunu indirip kurun
- Windows Defender'ın uygulamayı engellemediğinden emin olun

### Sürüm: 1.0.0
### Tarih: 12.07.2025
### Özellikler:
- Araç envanteri yönetimi
- Satış takibi ve istatistikleri
- Pazar trendleri analizi
- Windows uyumlu SQLite veritabanı
